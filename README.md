# Personal Finance Tracker

The **Personal Finance Tracker** is a desktop-based application developed using **Python (CustomTkinter)** and **MySQL**. It allows users to efficiently manage and monitor their financial activities, including accounts, transactions, budgets, and goals.

---

## Objective

The primary objective of this project is to provide users with an intuitive interface for tracking financial data, ensuring transparency and ease of management. It also integrates **MySQL triggers, functions, and procedures** to automate updates and maintain data consistency.

---

## Features

- **User and Admin Login:** Separate access panels for users and administrators.  
- **User Dashboard:** Enables users to view and manage accounts, transactions, budgets, and goals with full CRUD functionality.  
- **Admin Dashboard:** Allows administrators to view and manage all database tables.  
- **Automated Triggers:** Account balances are updated automatically upon new transactions.  
- **Scrollable Interface:** Tables and data views are neatly formatted and scrollable for ease of use.

---

## Technologies Used

| Component | Technology |
|------------|-------------|
| Programming Language | Python |

---

## Database Components

- **Trigger:** Updates account balance after each transaction.  
- **Function:** Retrieves account balance dynamically.  
- **Procedure:** Logs new expenses automatically.
